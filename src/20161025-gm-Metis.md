## 語意分析
可以從兩個方面來看：
1. 使用的字詞的意義
2. 字詞之間的關聯，包含相對位置與數目

先從把「看起來應該意義類似的語句」歸類在一起開始嘗試。
把看字詞數目相近的語句歸類成同一類。

## 開發狀況
1. 資料讀取
	- ` dict `
	- key 為評分準則的敘述
	- value 為評分者填寫的內容
2. 分詞
	- jieba
3. 詞頻
	- 得出同一平分準則下所有回應所使用的字詞
	- 計算每個回應中那些字詞的出現次數
	- 計算所有回應中那些字詞的出現總數
4. 向量化
	- 以不同字詞為維度，或不同文本（在這個 case 裡是語句）為維度
	- 決定資料在多維空間中的位置
5. Cluster
	- 以資料的位置來分群，進而決定哪些資料為同一類
